# 0x03-shell_variables_expansions

The **0x03-shell_variables_expansions** directory contains a collection of small projects focused on shell variables and expansions in the context of system engineering and DevOps. These projects cover concepts such as environment variables, aliases, arithmetic operations, and string manipulations using shell commands.
